# Documentação-SQL

Depois de muitos meses de estudos intensivos, revisões e vídeo-aulas, concluí uma documentação completa sobre o universo SQL.
Nela, abordo desde os conceitos iniciais até tópicos avançados, sempre com exemplos práticos e explicações objetivas.

Utilizei como base as principais documentações oficiais de SQL, além de cursos e materiais complementares.
Todo o conteúdo foi revisado, organizado e escrito de forma clara para facilitar o aprendizado.
